## Quick Start
1. Create a public repo named `username.github.io`.
2. Upload these files (keep folders as-is) and commit to `main`.
3. GitHub Actions will build & deploy.
4. Visit `https://username.github.io`.

## Edit
- `_config.yml`: name, email, links, nav.
- `index.md`: Home intro.
- `pages/research.md`, `pages/teaching.md`, `pages/cv.md`.
- `assets/`: upload `img/profile.jpg`, PDFs (e.g., `jmp.pdf`).

## Optional
- Replace CV page link with a local `/assets/cv.pdf` instead of Dropbox.
- Add Publications page if needed.
